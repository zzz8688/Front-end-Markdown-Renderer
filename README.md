字节跳动工程训练营前端营课题-实现⼀个前端Markdown渲染器                                                                                                                                                                                                                        
项目内容：这是一个基于React和TypeScript构建的前端Markdown实时渲染项目。其核心亮点是专为处理大语言模型（LLM）的流式文本而设计，通过自定义React Hook（useStreamingText）实现了平滑的流式渲染；采用强大的remark生态系统进行Markdown解析，并通过utils/markdownToHtml.ts文件中的逻辑将其转换为 HTML；此外，通过自定义remark插件remarkSpoiler.ts，扩展了标准Markdown语法，实现了类似“剧透”的特殊文本效果。整个应用的核心展示在pages/Playground中，它提供了一个交互式的界面，用户可以在一侧输入Markdown，另一侧的MarkdownPreview组件会立即进行实时、流式地渲染，直观地展示了该渲染器的核心能力。
